Reserve Your Seat

Archana Parasnis: parasnis.a@husky.neu.edu

Problem Statement: Provide a user-friendly website to allow users to book their seats for upcoming movie shows at theatres.

Description: The user (we will call him Buyer) should be able to anonymously find theatres near him, search for movies played at those theatres, see their details like date/time, cast etc. He should be able to book tickets for his preferred show. The admin user will be adding the theatre information, show details.

Domain objects: theatre, Movies, Show Details

A theatre plays many movies.

Human users: Admin, Anonymous user, Buyers, Agents

Admin: CRUD theatres, movies, show, users

Anonymous user: View/search theatre, movie and show details

Buyer: Same as anonymous user, apart from that he should be able to book his ticket for a show. Create reviews about the theatre/ movies. The user will need to login to create reviews/ follow other users. view the tickets he booked, cancel them

Agent: Same as anonymous user, apart from that he should be able to see deals on the tickets and book ticket for a show, view the tickets he booked, cancel them.
